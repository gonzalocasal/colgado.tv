PUBLICIDADES
Ignorar todas las publicidades que sugieran descargar un programa o ingresar un número de teléfono o cualquier otro dato personal. Este sitio no necesita ningún programa adicional, ni datos personales para funcionar.
BLOQUEAR PUBLICIDADES
Para bloquear las publicidades existe un plugin llamado "AdBlock Plus" disponible para Firefox - Opera - Chrome
Algunas transmisiones no funcionan cuando detectan esta extensión instalada.
